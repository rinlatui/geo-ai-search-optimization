# Contributing

This repository is a public GEO and AI Search Optimization framework.

## How to Contribute

You can contribute by improving:

- GEO checklists
- AI answer extraction examples
- Query fan-out templates
- Entity optimization workflows
- Local business GEO notes
- Multimodal GEO examples
- Reporting templates
- Testing rubrics

## Content Principles

- Keep recommendations practical.
- Avoid ranking guarantees.
- Avoid spam tactics.
- Add examples where useful.
- Separate proven SEO fundamentals from experimental AI visibility tracking.
- Do not include private client data, passwords, API keys, raw logs, or confidential URLs.

## Commit Style

Use clear commit messages:

```text
Add query fan-out mapping template
Improve author entity checklist
Update GEO testing rubric
```
