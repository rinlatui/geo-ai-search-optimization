# 07. Query Fan-out Coverage

Query fan-out coverage means planning content for the main query and the related sub-queries an AI system or user may need to answer fully.

## Query Types

- Main query
- Follow-up query
- Comparison query
- Definition query
- Decision query
- Problem-solving query
- Local query
- Commercial query
- Source verification query

## Example

Main query:

```text
SEO audit service
```

Fan-out queries:

```text
What is an SEO audit?
What does an SEO audit include?
How much does an SEO audit cost?
Technical SEO audit vs content audit
When should a business audit its website?
How to verify an SEO audit report?
```

## Output

```text
Main query:
Sub-queries:
Required sections:
Required examples:
Required sources:
Internal links:
```
