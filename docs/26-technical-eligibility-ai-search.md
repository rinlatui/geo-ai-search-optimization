# 26. Technical Eligibility for AI Search

Technical eligibility means a page can be crawled, indexed, rendered, and shown with a useful preview.

## Check

- URL returns 200.
- Page is indexable.
- Page is not blocked by robots.txt.
- Canonical is correct.
- Important content is crawlable.
- Important content is renderable.
- Snippet controls do not block useful previews.
- Page has enough visible content.
- Page works on mobile.

## Audit Questions

- Is this page eligible to appear in search?
- Is the correct canonical URL used?
- Are answer blocks visible in HTML/rendered content?
- Is the page blocked by any directive?
