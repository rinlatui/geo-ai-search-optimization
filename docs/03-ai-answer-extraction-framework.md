# 03. AI Answer Extraction Framework

AI answer extraction means making important information easy to identify, understand, and reuse as a concise answer.

## Extraction Formula

```text
Clear Question → Direct Answer → Evidence → Example → Verification
```

## Extractable Content Blocks

- Short answer block
- Definition block
- Step-by-step block
- Checklist block
- Comparison table
- Entity summary block
- FAQ block
- Source verification block

## Good Extraction Pattern

```text
[Question or topic]
Direct answer in 1-3 sentences.
Supporting explanation.
Example.
Verification method.
```

## Audit Questions

- Does the section answer a specific question?
- Can the answer stand alone?
- Is it written in plain language?
- Is the entity named clearly?
- Is the claim verifiable?
