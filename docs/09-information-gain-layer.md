# 09. Information Gain Layer

Information Gain is the new, useful, and specific value a page adds beyond generic content.

## Information Gain Formula

```text
Information Gain = Experience + Specificity + Original Structure + Verification + Practical Use
```

## What Adds Information Gain

- Original framework
- Real-world observations
- Specific examples
- Decision criteria
- Mistake analysis
- Checklists
- Templates
- Data or source support
- Updated context
- Clear verification methods

## Audit Questions

- What does this page add that similar pages do not?
- Is there practical experience?
- Are examples specific?
- Is the advice actionable?
- Can users verify important claims?
