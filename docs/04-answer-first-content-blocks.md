# 04. Answer-First Content Blocks

Answer-first content starts with the direct answer before deeper explanation.

## Why It Matters

AI search systems and users both benefit from clear answers near the beginning of a section.

## Template

```text
[Topic] is [direct definition or answer].

It matters because [reason].
Use it when [condition].
Avoid it when [condition].
```

## Example

```text
GEO is the process of optimizing content so AI search systems can understand, extract, summarize, and cite information from a website more accurately.
```

## Checklist

- [ ] First sentence answers the query
- [ ] Answer is not vague
- [ ] Entity is named
- [ ] No unnecessary introduction
- [ ] Explanation follows after the answer
