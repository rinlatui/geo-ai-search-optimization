# 35. GEO Reporting Dashboard

A GEO reporting dashboard tracks visibility, citations, entity mentions, and content actions.

## Dashboard Fields

- Date
- Query
- Engine
- Page / URL
- Entity mentioned
- URL cited
- Competitor cited
- Answer accuracy
- Missing information
- Action required
- Owner
- Status

## Reporting Rhythm

- Weekly for important campaigns
- Monthly for content hubs
- Quarterly for entity/source graph review

## Dashboard Output

```text
Total queries tested:
Mentions:
Citations:
Incorrect answers:
Competitor citations:
Pages to update:
Entity issues:
Next actions:
```
