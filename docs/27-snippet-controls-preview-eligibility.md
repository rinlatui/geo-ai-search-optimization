# 27. Snippet Controls and Preview Eligibility

Snippet controls affect how much content can be displayed in search previews.

## Controls to Review

- `nosnippet`
- `max-snippet`
- `data-nosnippet`
- `noindex`
- X-Robots-Tag
- Meta description
- Visible answer blocks

## Risk

If snippet controls are too restrictive, the page may be less eligible for useful previews in search experiences.

## Audit Questions

- Does the page have `nosnippet`?
- Is `max-snippet` too short?
- Is important content wrapped in `data-nosnippet`?
- Is X-Robots-Tag restricting snippets?
- Does the meta description accurately summarize the page?
