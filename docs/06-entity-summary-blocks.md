# 06. Entity Summary Blocks

Entity summary blocks help search engines and AI systems understand who or what the page is about.

## Template

```text
Entity:
Type:
Primary URL:
Related entities:
Attributes:
Known for:
sameAs:
```

## Example

```text
Entity: Doan Trinh Duc
Type: Person
Primary URL: https://seobeginer.net/author/doan-trinh-duc/
Related entities: TD Digital, SEOBeginer, W3SEO
Known for: Technical SEO, WordPress SEO, EEAT, schema, content architecture
sameAs: GitHub, LinkedIn, author profiles
```

## Checklist

- [ ] Entity name is consistent
- [ ] Entity type is clear
- [ ] Primary URL is clear
- [ ] Related entities are relevant
- [ ] sameAs links are valid
- [ ] Schema matches visible content
