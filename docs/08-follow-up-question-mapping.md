# 08. Follow-up Question Mapping

Follow-up question mapping helps a page answer the next questions users are likely to ask.

## Process

1. Identify the main query.
2. List likely follow-up questions.
3. Group questions by intent.
4. Map each group to H2/H3 sections.
5. Add FAQ for short direct answers.
6. Add internal links for deeper topics.

## Question Groups

| Group | Example |
|---|---|
| Definition | What is it? |
| Process | How does it work? |
| Comparison | A vs B |
| Cost | How much does it cost? |
| Risk | What can go wrong? |
| Decision | When should I choose it? |
| Verification | How can I check it? |

## Checklist

- [ ] Follow-up questions are mapped
- [ ] Questions are grouped by intent
- [ ] Important questions have visible answers
- [ ] FAQ is not fake or repetitive
- [ ] Deeper answers link to supporting pages
