# 23. Search Console and AI Visibility Notes

Search Console remains essential for understanding indexing, clicks, impressions, crawling, sitemaps, and Core Web Vitals.

## What to Use Search Console For

- Indexing status
- URL inspection
- Sitemap status
- Query impressions
- Click trends
- Page performance
- Core Web Vitals
- Manual actions
- Security issues

## AI Visibility Notes

Search Console data can help identify pages that already receive search visibility, but AI visibility may require additional manual or tool-based tracking depending on available reports and markets.

## Audit Workflow

1. Identify important pages.
2. Check indexing status.
3. Review queries and pages.
4. Compare with AI visibility manual tests.
5. Update pages with missing answer/entity/source blocks.
6. Track changes over time.
