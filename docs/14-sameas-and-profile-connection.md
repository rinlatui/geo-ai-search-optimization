# 14. sameAs and Profile Connection

sameAs links help connect an entity to its external profiles and identity references.

## Recommended sameAs Targets

For a person:

- Main author profile
- GitHub
- LinkedIn
- Professional social profiles
- Other author pages
- Organization profile

For an organization:

- Official website
- Google Business Profile where appropriate
- LinkedIn company page
- Facebook page
- YouTube channel
- Public profiles

## Quality Rules

- Only use real profiles.
- Do not add fake profiles.
- Keep names consistent.
- Ensure URLs return valid pages.
- Keep visible content aligned with schema.

## Checklist

- [ ] sameAs links are visible or supported by page content
- [ ] sameAs URLs are correct
- [ ] sameAs URLs return 200
- [ ] Entity names match
- [ ] Schema is not misleading
