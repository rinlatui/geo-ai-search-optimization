# 19. Image and Video GEO Signals

Multimedia content helps users and AI systems understand complex topics when it is properly supported.

## Image Signals

- Descriptive filename
- Descriptive alt text
- Caption when useful
- Surrounding text
- Image context
- Compressed file size
- Original visuals where possible

## Video Signals

- Title
- Description
- Transcript
- Chapters
- Captions
- Supporting article text
- Video schema where appropriate

## Audit Questions

- Can the image be understood from alt/caption/context?
- Does the video have a transcript?
- Does the page provide text support for visual content?
