# 16. FAQ and Comparison Optimization

FAQ and comparison blocks help answer follow-up questions and decision queries.

## FAQ Rules

- Answer real questions.
- Keep answers direct.
- Avoid fake or repetitive FAQ.
- Make FAQ visible on the page.
- Add schema only when it matches visible content.

## Comparison Rules

Use tables for:

- A vs B
- Service comparison
- Tool comparison
- Pricing tiers
- Pros and cons
- Use cases

## Example Table

| Option | Best For | Avoid When |
|---|---|---|
| Technical SEO audit | Crawl/index issues | You only need copywriting |
| Content audit | Content quality issues | The site has crawl errors |
