# 11. Source and Verification Layer

Source and verification signals help users and AI systems understand whether information can be trusted.

## Use Verification For

- Technical claims
- Legal or policy claims
- Medical/financial/security topics
- Product specifications
- Data/statistics
- Standards
- Search engine documentation
- Time-sensitive claims

## Verification Methods

- Official documentation
- Tool screenshots
- Public source links
- Step-by-step checking method
- Last updated date
- Author/reviewer notes
- Version history

## Template

```text
Claim:
Source:
Verification method:
Last checked:
Limitations:
```
