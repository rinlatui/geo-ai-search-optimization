# 31. Multimodal GEO: Image / Video / Visual Search

AI search experiences may use text, images, videos, and structured product or local information.

## Image Optimization

- Original image where possible
- Descriptive filename
- Useful alt text
- Caption where useful
- Context around the image
- Compressed format
- Relevant dimensions

## Video Optimization

- Clear title
- Description
- Transcript
- Chapters
- Captions
- Supporting article
- Video schema where appropriate

## Visual Search Questions

- Can the image be understood without seeing the full page?
- Does the page describe what the visual shows?
- Is there text support for infographics?
- Are videos supported by transcripts?
