# 28. Anti-Hack / GEO Mythbusting

GEO should not become a spam tactic.

## Avoid

- Fake mentions
- Fake reviews
- Fake author profiles
- Schema spam
- Fake FAQ
- Keyword stuffing
- Mass-generated low-value pages
- Claiming guaranteed AI Overview visibility
- Treating llms.txt as a required Google Search ranking file
- Hiding text only for AI systems

## Do Instead

- Create useful content for people.
- Make answers clear.
- Connect real entities.
- Support claims with sources.
- Keep schema accurate.
- Track visibility honestly.
