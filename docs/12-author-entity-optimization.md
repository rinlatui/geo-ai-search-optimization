# 12. Author Entity Optimization

Author entity optimization connects content to a clear, verifiable person.

## Author Signals

- Author page
- Bio
- Profile photo
- Job title
- Expertise topics
- WorksFor relationship
- GitHub / LinkedIn / social profiles
- sameAs links
- Published articles
- Reviewer role where appropriate

## Audit Questions

- Is the author page indexable?
- Is the author name consistent?
- Is the bio specific?
- Is GitHub or LinkedIn connected?
- Does Person schema match visible content?
- Are other author pages connected to the main profile?
