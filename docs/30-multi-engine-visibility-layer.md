# 30. Multi-Engine Visibility Layer

GEO should not be measured only in one search interface.

## Engines / Surfaces to Track

- Google Search AI features
- Bing / Copilot search experiences
- ChatGPT Search
- Perplexity
- Gemini
- Featured snippets
- People Also Ask
- Video/image search surfaces

## Tracking Fields

- Query
- Engine
- Date
- Entity mentioned
- URL cited
- Competitor cited
- Answer accuracy
- Missing information
- Next action

## Important Note

Different engines may use different sources and formats. Track patterns, not one-time results.
