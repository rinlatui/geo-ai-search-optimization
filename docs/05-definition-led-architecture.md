# 05. Definition-Led Architecture

Definition-led architecture means important sections begin with a clear definition before examples, stories, or details.

## Use For

- Concepts
- Services
- Technical terms
- Processes
- Tools
- Frameworks
- Entities

## Definition Formula

```text
[Term] is [category] that [function] for [user/outcome].
```

## Example

```text
Technical SEO is the process of improving a website's crawlability, indexability, performance, structured data, and technical quality so search engines can access and understand important pages.
```

## Audit Questions

- Is the term defined before it is discussed?
- Is the definition specific?
- Does it explain function and context?
- Does it avoid keyword stuffing?
