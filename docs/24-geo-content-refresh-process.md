# 24. GEO Content Refresh Process

GEO content needs refreshing when search intent, AI answers, facts, competitors, or source expectations change.

## Refresh Triggers

- Traffic drop
- Query shift
- AI answer misinformation
- Competitor source appears repeatedly
- Outdated facts
- Missing entity signals
- Weak Information Gain
- Weak answer extraction

## Refresh Workflow

1. Re-check search intent.
2. Re-map query fan-out.
3. Add answer-first blocks.
4. Add missing definitions.
5. Improve entity summary.
6. Add verification/source layer.
7. Improve internal links.
8. Re-test visibility.
