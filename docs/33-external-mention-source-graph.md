# 33. External Mention and Source Graph

A source graph maps public references that support entity understanding.

## Source Graph Elements

- Main website
- Author profile
- GitHub
- LinkedIn
- Organization website
- Public case studies
- Public repositories
- Press mentions
- Professional profiles
- Other author pages

## Quality Rules

- Use real sources.
- Avoid fake mentions.
- Avoid irrelevant directories.
- Keep names consistent.
- Connect profiles naturally.
- Maintain profiles over time.

## Output

```text
Entity:
Primary source:
Supporting sources:
Broken links:
Inconsistencies:
Next actions:
```
