# 10. Fact Density Layer

Fact density measures how much useful, specific, and verifiable information appears in a section.

## High Fact Density Pattern

```text
Definition → Specific detail → Example → Condition → Verification
```

## Low Fact Density Signs

- Generic claims
- Long paragraphs with little meaning
- Repeated ideas
- No examples
- No verification
- No decision support

## Improvement Methods

- Add numbers where appropriate
- Add conditions
- Add examples
- Add checklists
- Add source references
- Remove filler phrases

## Checklist

- [ ] Each section has a clear answer
- [ ] Claims are specific
- [ ] Examples are included
- [ ] Verification is possible
- [ ] Filler text is removed
