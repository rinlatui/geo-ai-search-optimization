# 15. Structured Data for GEO

Structured data can help search systems understand entities, page types, breadcrumbs, articles, FAQs, products, and organizations.

## Useful Schema Types

- Person
- Organization
- LocalBusiness
- WebSite
- WebPage
- Article
- FAQPage
- BreadcrumbList
- Product / Offer where appropriate
- ItemList where appropriate

## GEO Rule

Schema should support visible content, not replace it.

## Audit Questions

- Does schema match visible content?
- Is Person schema consistent?
- Is Organization schema consistent?
- Does FAQ schema match visible FAQ?
- Does BreadcrumbList match real breadcrumbs?
- Are there duplicate conflicting schema graphs?
