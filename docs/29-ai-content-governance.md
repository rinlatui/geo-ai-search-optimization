# 29. AI Content Governance

AI can assist content work, but governance is needed to avoid low-value, inaccurate, or scaled abuse content.

## Governance Principles

- Human review is required.
- Claims must be checked.
- Experience-based insight should be added.
- Sources should be verified.
- Sensitive topics need extra caution.
- Do not publish mass-generated thin pages.
- Keep version history when useful.
- Add disclosures when users would reasonably expect them.

## AI Content Workflow

```text
Research support → Outline → Draft → Human editing → Fact check → SEO/GEO review → Publish → Refresh
```
