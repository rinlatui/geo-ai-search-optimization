# 34. GEO Testing and Evaluation Rubric

A rubric makes GEO review measurable.

## Scoring Categories

Score each category from 0 to 10.

| Category | Score |
|---|---|
| Answer Extraction | 0-10 |
| Entity Clarity | 0-10 |
| Information Gain | 0-10 |
| Fact Density | 0-10 |
| Source Verification | 0-10 |
| Author Trust | 0-10 |
| Organization Trust | 0-10 |
| Technical Eligibility | 0-10 |
| Snippet Eligibility | 0-10 |
| Structured Data Quality | 0-10 |
| Internal Linking | 0-10 |
| Multimodal Support | 0-10 |
| Multi-Engine Visibility | 0-10 |
| Reporting Readiness | 0-10 |

## Interpretation

```text
0-60: Weak GEO readiness
61-90: Needs improvement
91-115: Good foundation
116-140: Strong GEO-ready content
```
