# 02. AI Search vs Traditional Search

Traditional search often returns a ranked list of documents. AI search experiences may summarize, compare, synthesize, and cite information from multiple sources.

## Traditional Search Focus

- Ranking position
- Title and snippet
- Click-through rate
- Query-to-URL matching
- Indexed pages
- Backlinks and internal links

## AI Search Focus

- Extractable answers
- Source trust
- Entity clarity
- Query fan-out coverage
- Consistency across sources
- Original information
- Citation-readiness
- Multi-format support

## Practical Difference

Traditional SEO asks:

```text
Can this page rank for this query?
```

GEO asks:

```text
Can this page help an AI system answer this query accurately, and can it be recognized as a trustworthy source?
```
