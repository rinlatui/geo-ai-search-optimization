# GEO Page Review Workflow

1. Check technical eligibility.
2. Check snippet controls.
3. Identify main query and entity.
4. Review answer-first block.
5. Review definition-led sections.
6. Review Information Gain.
7. Review source verification.
8. Review author and organization signals.
9. Review internal links.
10. Score the page using the GEO rubric.
11. Record actions in the dashboard.
