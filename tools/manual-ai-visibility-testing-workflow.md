# Manual AI Visibility Testing Workflow

## Goal

Track whether a brand, entity, URL, or source appears in AI-assisted search experiences.

## Steps

1. Select a query.
2. Select an engine.
3. Run the query in a clean session where possible.
4. Record whether the entity is mentioned.
5. Record whether the URL is cited.
6. Record competitors cited.
7. Record incorrect or missing information.
8. Decide what content block needs improvement.
9. Re-test after the update.

## Recommended Fields

- Date
- Query
- Engine
- Entity mentioned
- URL cited
- Competitor cited
- Answer accuracy
- Issue
- Next action
