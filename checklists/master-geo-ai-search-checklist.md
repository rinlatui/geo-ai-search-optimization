# Master GEO AI Search Checklist

## Technical Eligibility

- [ ] Page returns 200
- [ ] Page is indexable
- [ ] Canonical is correct
- [ ] Important content is crawlable/renderable
- [ ] Snippet controls are not overly restrictive

## Answer Extraction

- [ ] Page has answer-first blocks
- [ ] Important terms have definitions
- [ ] Steps are numbered where useful
- [ ] Comparisons use tables
- [ ] FAQ answers are direct

## Entity and Trust

- [ ] Main entity is clear
- [ ] Author entity is clear
- [ ] Organization entity is clear
- [ ] sameAs links are valid
- [ ] Sources or verification methods are included

## Information Quality

- [ ] Information Gain is present
- [ ] Fact density is strong
- [ ] Claims are specific
- [ ] Examples are practical
- [ ] Filler content is removed

## Measurement

- [ ] AI visibility tracking template is prepared
- [ ] Query fan-out map is prepared
- [ ] Reporting dashboard is updated
- [ ] Refresh date is scheduled
