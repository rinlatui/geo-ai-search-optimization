## What GEO Means Here

In this project, GEO means **Generative Engine Optimization**: a practical approach to making content easier for AI search systems to understand, extract, summarize, verify, and cite.

This is not a hack. This framework is based on:

```text
Helpful content
+ Technical eligibility
+ Clear entity signals
+ Extractable answer structure
+ Information Gain
+ Verification
+ Measurement
```

## Quick Start

1. Read `docs/01-geo-fundamentals.md`
2. Build a content plan with `templates/geo-content-brief-template.md`
3. Map query fan-out with `templates/query-fanout-map-template.csv`
4. Create answer blocks with `templates/ai-answer-block-template.md`
5. Audit entity clarity with `checklists/author-entity-optimization-checklist.md`
6. Score the page with `checklists/geo-evaluation-rubric.md`
7. Track results with `templates/geo-reporting-dashboard-template.csv`
